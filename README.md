# 🦀 Rust Zone

Landing page responsiva inspirada em um servidor de Rust, desenvolvida com o objetivo de praticar e consolidar meus conhecimentos em HTML e CSS.

## 💻 Sobre o projeto

O Rust Zone é uma landing page fictícia para um servidor de Rust.

O projeto foi desenvolvido durante meus estudos de desenvolvimento web, com foco na criação de layouts, organização de elementos com Flexbox e adaptação da página para diferentes tamanhos de tela.

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- Flexbox
- Media Queries
- Font Awesome

## 📚 O que pratiquei

Durante o desenvolvimento deste projeto pratiquei:

- Estruturação semântica com HTML
- Organização de layouts utilizando Flexbox
- Box Model (margin, padding e dimensões)
- Seletores CSS
- Pseudo-classe `:hover`
- Utilização de ícones com Font Awesome
- Responsividade com Media Queries
- Adaptação do layout para desktop, tablet e mobile

## 📱 Responsividade

A página foi desenvolvida para se adaptar a diferentes tamanhos de tela, incluindo desktop, tablet e dispositivos móveis.
